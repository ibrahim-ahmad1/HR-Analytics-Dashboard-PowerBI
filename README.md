# 📊 HR Analytics Dashboard (Power BI)

## 🔍 Overview

This project analyzes employee data to understand attrition trends, salary distribution, and workforce demographics.

## 🎯 Objectives

* Analyze employee attrition
* Identify key factors affecting attrition
* Provide insights for HR decision-making

## 📌 Key Insights

* Total Employees: 1470
* Attrition Count: 237
* Attrition Rate: 16.1%
* Highest attrition in age group: 26–35
* Most affected roles: Laboratory Technician, Sales Executive

## 📊 Dashboard Features

* Attrition by Age, Salary, Education
* Department-wise filtering
* Job role analysis
* Interactive visuals

## 🛠 Tools Used

* Power BI
* Excel (Dataset)
